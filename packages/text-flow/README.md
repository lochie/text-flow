# text-flow

An animated text component for React.

## Installation

```shell
pnpm i text-flow
```

## Usage

```tsx
import TextFlow from "text-flow";

<TextFlow>Hello world</TextFlow>;
```

## Options

```tsx
<TextFlow
  children="Hello world"
  duration={1} // duration of the transition in seconds
/>
```
