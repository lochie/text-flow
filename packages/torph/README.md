# torph

An animated text component for React.

## Installation

```shell
pnpm i torph
```

## Usage

```tsx
import { TextMorph } from "torph";

<TextMorph>Hello world</TextMorph>;
```

## Options

```tsx
<TextMorph
  children="Hello world"
  duration={1} // duration of the transition in seconds
/>
```
