# Torph

An animated text component for React.

## Installation

```shell
pnpm i torph
```

## Usage

```tsx
import { TextMorph } from "torph";

<TextMorph>Hello world</TextMorph>;
```

# Contributing

## Install dependencies

```sh
pnpm install:all
```

## Dev/Watch Library and Example

```sh
pnpm dev
```

## Build Library

```sh
pnpm build
```
